Nimipay

Simple and Flexible Payments Gateway

Add Nimipay to any website and start accepting cryptocurrency, and indirectly - credit card payments, anonymously and effortlessly









Examples

Below are four examples showing different types of Nimipay integrations.
Try to open a wallet and buy some fortune cookies.





Open My NIM Wallet
<div onclick="npWallet()" class="np-btn">Open My NIM Wallet</div>

The above code will place a button on your website that will allow the user to open his or her NIM wallet or to create a new one.




Fortune Cookie (0.01$)
Add to Cart
<div onclick="npAddItem()" data:type="fortune_cookie" data:value="0.1" data:meta="" class="np-btn">Add to Cart</div>

When clicked, the item's invoice will be added to the user's wallet where it can be paid for, and then the item received.




Fortune Cookie (0.01$)
Instant Buy
<div onclick="npAddItemCheckout()" data:type="fortune_cookie" data:value="0.1" data:meta="instant" class="np-btn">Instant Buy</div>

Single invoice, instant pay. When clicked, it will open a Nimiq checkout screen with the new invoice to pay. The meta tag is optional. It can be used to pass additional data to the backend, e.g. item ID. For the above cookie example, invoice and item IDs are generated on the backend.





Donate
<div onclick="npDonate()" data:msg="Donation to Nimipay" class="np-btn">Donate</div>

Instant checkout screen without the invoice and UI. The user can choose a custom NIM value and then send the transaction.






Quick Start

Initialize Nimipay (~30 kb) on any website

<script>
const nimAddress = 'NQ13 YOUR WALL ETAD DRES S1F9 T4L7 EA13 N7MP';
const nimAddressLabel = 'Your Brand Name';
</script>

<script src="https://nimipay.com/np.0.0.9.min.js"></script>
<link href="https://nimipay.com/np.0.0.9.min.css" rel="stylesheet">


Out of the box, Nimipay works client-side and without the backend. Interactions with the blockchain return data to the website, which then can be passed to the backend. See the source of a lightweight PHP and MySQL backend used on this site.

Nimipay is open source. You can customize it to your individual needs.

Why Nimipay

• Nimipay is lightweight, less than 30 kb JS library
• One minute install on any website, no coding/crypto skills needed
• Simple to create a new NIM wallet or open an existing one
• With shortcodes, it is easy to invoice your users
• Accept payments from your users in NIM cryptocurrency
• Users can use bank cards to quickly "top up" their NIM wallets
• Privacy-focused, client-side, open-source, decentralized
• Works on any device with an internet browser
• Global and borderless payments


Nimipay is the first service that allows websites to accept payments effortlessly, and with inherent security and privacy.

It is possible only because it is based on Nimiq blockchain. The only blockchain that is in your browser and requires no installation. Nimiq is Crypto 3.0 - it provides the level of simplicity that can be intuitively grasped by any layman. Anyone can create and secure a Nimiq wallet in seconds. Furthermore, Nimiq is based on the essential cryptocurrency values and principles (security, decentralization, client-side key management, PoS).

Powered by the browser-based Nimiq blockchain, Nimipay allows any website to accept cryptocurrency and credit card payments effortlessly and anonymously.

Additionally, by giving each user a cryptocurrency wallet, custom solutions can be built on top of Nimipay, e.g. your users can deposit NIM to your website and then withdraw them. You can also pay your users by distributing affiliate earnings, airdrops, bounties, rewards, etc. NIM can be used as a currency in a virtual economy, e.g. in a browser-based game, or attached to the checkout flow in a traditional webshop.

Nimipay is open source and does not charge any fees for the service.
